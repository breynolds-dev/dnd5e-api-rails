Introduction

There are a few things of note before we begin:

info This API is consumption-only, thus the only HTTP method we accept is a GET request.

info There is No Authentication at this time. This is free for public use and consumption, be aware that this could change in the future as our server loads increase

Wrappers

At this time we do not have any wrappers for our API. If you would like to create one we would love to see it and get it added here!


Races

The worlds of Dungeons & Dragons have many different races to choose from, each with their own unique characteristics and traits. There are several endpoint options to choose from in order to get race information from the API.


GET v1/races

                

              

              
The first endpoint you can target will return an array of all of the race options currently in our API

              

                
                  

                    Name
                    Description
                    Data Type
                  

                
                
                  

                    name
                    The name of the class resource
                    string
                  

                  

                    url
                    The API endpoint url to the associated resource
                    string
                  

                
              

            


            

            
GET v1/races/{race}

            

              

                

                  

                

              

              

                
Attempting to access a class resource when no subclasses are available will return a JSON object for the associated class.

                

                  
                    

                      Name
                      Description
                      Data Type
                    

                  
                  
                    

                      name
                      The name of the class resource
                      string
                    

                    

                      subrace
                      The characters subrace if applicable
                      string
                    

                    

                      description
                      A collection of information describing the history and various details about a race
                      array/string
                    

                    

                      speed
                      The amount of ground a race can cover in one action, measured in feet
                      integer
                    

                    

                      darkvision
                      Distance a race can see in the dark, measured in feet
                      integer
                    

                    

                      stat_bonus
                      The ability bonuses the specefic race is granted by default
                      CollectionResource (ability)
                    

                    

                      skills
                      Any skills that the race gains proficiency in by dafault
                      CollectionResource (skill)
                    

                    

                      extra_skill_proficiencies
                      Represents the number of extra skills a race can gain
                      integer
                    

                    

                      weapon_proficiencies
                      An array of the weapon proficiencies granted to a race
                      array/sting
                    

                    

                      armor_proficiencies
                      An array of the armor proficiencies granted to a race
                      array/string
                    

                    

                      min_age
                      The maximum age before a character of that race is mature
                      integer
                    

                    

                      max_age
                      The maximum age before a character of that race is considered elder
                      integer
                    

                    

                      age_description
                      A short description about how a race ages
                      string
                    

                    

                      size
                      The size type of the associated race
                      string
                    

                    

                      size_description
                      A short description covering the size information for the associated race
                      string
                    

                    

                      min_height
                      The minimum height for a character of that race
                      integer
                    

                    

                      max_height
                      The maximum height for a character of that race
                      integer
                    

                    

                      min_weight
                      The minimum weight for a character of that race
                      integer
                    

                    

                      max_height
                      The maximum weight for a character of that race
                      integer
                    

                    

                      languages
                      A collection of known languages that a character of the given race can speak by default
                      CollectionResource (langauge)
                    

                    

                      traits
                      A collection of racial traits gained by default
                      CollectionResource (trait)
                    

                    

                      links
                      A collection of links
                      CommonResource (links)
